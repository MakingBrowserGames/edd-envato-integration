How to Install

Installing the plugin is very simple:

1. Go to your WordPress dashboard and add a new plugin.

2. Use the upload option and select the edd-envato.zip file NOT the file you downloaded from the marketplace!

3. Activate and continue on to the settings page.

Plugin Setup / Settings

Uder Settings -> Edd Envato

On this page you will setup the plugin and the way it functions as well as configure your discount options.

Enable EDD Envato - Here you can quickly enable / disable the plugin.

Envato Username and API Key - This must be YOUR marketplace username and API key you want to validate purchase codes against. If these are not valid, the plugin will not function.

Redirect Page - Here you can select the page you want to redirect the user to after the registration is complete.

Verify Code Message - Here you can enter the message that will appear over the purchase code verification form.

 

Discount Settings

Discount Enabled - If enabled your users will get an email containing a coupon code they can use ( single use ) when they register.

Discount Amount & Type - Here you can enter the amount and type of discount to use.

Discount Email Subject - The subject line of the email sent to the user.

Discount Email Message - This is the message that will appear over the discount code in the email.

 

Envato Purchase Codes

Under Users -> Purchase codes you can also view a table of all the purchase codes used to create accounts on your site. Fully searchable of course.

How to Use

Using the plugin is very simple, anywhere you insert the [edd-envato] shortcode the form will appear all set up. That's it! There is also a built in tinyMCE toolbar button that will one-click insert the shortcode for you from the post editor. Everything else will be done for you.

Note that you must be using the shortcode in an area that will render it otherwise you will need to use the do_shortcode() function.
